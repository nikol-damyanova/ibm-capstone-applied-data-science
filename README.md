# IBM Apllied Data Science Capstone Project

Methodologies and Tools:
- Python
- Data Collection (web scraping wth beautiful soup)
- Data Wrangling
- Exploratory Data Analysis in SQL
- Interactive Visual Analytics with Folium, Plotly
- Predictive Analysis using Scikit\SkLearn

This project uses Machine Learning to calculate where Falcon 9 lands, correlate it to the total cost and determine the cost of launch.
